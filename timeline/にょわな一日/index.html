業務ではブランドサイトを新たに構築するため、サイトのデザイン案をあれこれ作っていた。 事業所ではAdobe Creative Cloudの製品群が一通り利用できるため、初めてAdobe XDを触ってみた。

hugoの操作コマンド

（主に自分用として）hugoの操作コマンドについて簡潔にまとめる。

hugo new site hogehoge　//ホームファイルを作る

hugo new hogehoge.md //新しい.md記事をcontentフォルダに作成する

hugo server //サーバーを起動する

hugo //　public配下にHTML類を生成する（ビルド）
